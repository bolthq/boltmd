# ⚡ Welcome to BoltMD

> The fastest Markdown editor on the planet. Period.

---

## Why BoltMD?

We got tired of waiting.

Waiting for editors to launch. Waiting for files to load. Waiting for the cursor to catch up with our thoughts. So we built something that doesn't make you wait. **Ever.**

BoltMD opens in the time it takes you to blink. It renders Markdown as fast as you can type it. And it does all this in a package smaller than a single Spotify song.

**This is what a Markdown editor should have been all along.**

## ✨ What Makes Us Different

### ⚡ Absurdly Fast
Launch in under 500ms. That's not a typo — half a second from click to cursor. Most editors are still loading their splash screen by the time you're already writing in BoltMD.

### 🪶 Impossibly Light
Under 15MB installed. Under 100MB RAM while running. We squeezed every last byte because your computer has better things to do than babysit a text editor.

### 👁️ Beautiful WYSIWYG
Real-time rendering that rivals Typora — headings, code blocks, tables, math formulas, all rendered live as you type. No split panes. No awkward preview windows. Just beautiful text, instantly.

### 📝 Raw Power When You Need It
One keystroke switches to source mode with a world-class code editor (the same engine that powers VS Code). Syntax highlighting, multi-cursor, minimap — the works.

### 🗂️ Multi-Tab Like a Pro
Open ten files at once. Drag tabs around. Reference your notes while drafting your masterpiece. We're a text editor, not a one-trick pony.

### 🌓 Easy on the Eyes
Light mode. Dark mode. Switch in a heartbeat. Because great writing happens at 2 AM too.

### 📋 Paste Anything
Screenshots from your clipboard? HTML from the web? Images dragged from your desktop? We handle it all. Just paste — we'll figure out the rest.

### 🔒 100% Private
No cloud. No accounts. No telemetry. Your files live on your computer and nowhere else. We literally can't read your documents even if we wanted to.

## 🏗️ Under the Hood

We didn't cut corners on the tech stack either:

| Layer | Technology | Why |
|-------|-----------|-----|
| Shell | **Tauri 2.0** | 10x smaller than Electron, native OS integration |
| Editor | **Tiptap + CodeMirror 6** | Best-in-class editing engines, battle-tested |
| Frontend | **Vue 3 + TypeScript** | Reactive, type-safe, blazing fast |
| Backend | **Rust** | Memory-safe, zero-cost abstractions, native speed |

**Translation:** We used the fastest tools available and made them work together seamlessly. The result speaks for itself.

## 📂 Make BoltMD Your Default

Right-click any `.md` file → **Open with** → choose **BoltMD** → check "Always use this app."

That's it. Every Markdown file on your computer now opens at lightning speed.

`Supported:` `.md` · `.markdown` · `.mdown` · `.mkd` · `.mkdn`

## 💡 Pro Tips

1. **Drag & drop** any file onto the window — we'll open it instantly
2. **Paste images** straight from clipboard — saved alongside your document automatically
3. **`Ctrl + /`** is your best friend — seamlessly switch between WYSIWYG and source mode
4. **Multiple tabs** + drag to reorder = your personal writing cockpit
