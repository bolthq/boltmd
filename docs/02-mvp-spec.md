# BoltMD — MVP 功能规格说明书 (v0.1)

## MVP 目标

用户可以双击 .md 文件，BoltMD 在 500ms 内打开并显示格式化的 Markdown 内容，支持所见即所得编辑和源码编辑，保存后是标准 Markdown 文本。

## 功能清单

### P0 — 核心（没有这些不能发布）

#### F01: 文件操作

| 功能 | 描述 | 快捷键 |
|------|------|--------|
| 新建文件 | 创建空白 .md 文件 | Ctrl+N |
| 打开文件 | 系统文件对话框，筛选 .md/.markdown/.txt | Ctrl+O |
| 保存 | 保存到原路径，新文件弹出另存为 | Ctrl+S |
| 另存为 | 选择路径保存 | Ctrl+Shift+S |
| 命令行打开 | `boltmd file.md` 直接打开 | — |
| 拖拽打开 | 拖拽文件到窗口打开 | — |
| 文件类型关联 | .md/.markdown 双击用 BoltMD 打开 | — |

**验收标准：**
- 打开 1MB 的 .md 文件 < 1 秒
- 保存不阻塞编辑（异步写入）
- 文件编码自动检测（UTF-8/GBK/UTF-16）
- 未保存时关闭窗口弹出确认对话框

#### F02: WYSIWYG 编辑模式

| Markdown 语法 | 编辑器行为 |
|---------------|-----------|
| `# ~ ######` 标题 | 输入 `# ` + 空格后自动转为标题样式 |
| `**bold**` | 加粗文本实时渲染 |
| `*italic*` | 斜体实时渲染 |
| `` `code` `` | 行内代码高亮 |
| `> quote` | 引用块样式 |
| `- / * / 1.` 列表 | 有序/无序列表自动缩进 |
| `- [ ] / - [x]` | 任务列表，可点击切换 |
| `---` 分隔线 | 水平线渲染 |
| ` ``` ` 代码块 | 语法高亮的代码块 |
| `[text](url)` 链接 | 可点击链接，编辑时显示 Markdown 语法 |
| `![alt](url)` 图片 | 内联显示图片 |
| `| 表格 |` | 表格渲染，可编辑单元格 |

**验收标准：**
- 通过 [CommonMark 规范](https://commonmark.org/) 测试
- 输入响应延迟 < 16ms（60fps）
- 支持中文输入法（IME composing 状态正确处理）
- 撤销/重做（Ctrl+Z / Ctrl+Shift+Z）
- 复制粘贴保留格式

#### F03: 源码编辑模式

| 功能 | 描述 |
|------|------|
| 语法高亮 | Markdown 语法着色 |
| 行号 | 可配置显示/隐藏 |
| 自动缩进 | 列表、代码块自动缩进 |
| 括号匹配 | 括号/引号自动配对 |
| 搜索替换 | Ctrl+F / Ctrl+H |
| 代码折叠 | 按标题层级折叠 |
| 自动换行 | 可配置 |

**验收标准：**
- 1 万行文件编辑不卡顿
- 搜索 10 万字符文件 < 100ms

#### F04: 分屏预览模式

| 功能 | 描述 |
|------|------|
| 左右分屏 | 左边源码，右边 Tiptap 只读模式预览（保证与 WYSIWYG 渲染一致） |
| 同步滚动 | 编辑区和预览区滚动同步 |
| 实时更新 | 编辑时预览实时刷新（防抖 200ms） |

#### F05: 模式切换

| 功能 | 描述 | 快捷键 |
|------|------|--------|
| 切换编辑模式 | WYSIWYG ↔ 源码 ↔ 分屏，循环切换 | Ctrl+/ |
| 内容同步 | 切换时内容无损传递 | — |
| 光标恢复 | 切换后光标位置尽量保持一致 | — |
| 记忆模式 | 记住用户上次使用的模式 | — |

#### F06: 图片支持

| 功能 | 描述 |
|------|------|
| 粘贴图片 | Ctrl+V 粘贴剪贴板图片，自动保存到 `./assets/` |
| 拖拽图片 | 拖拽图片文件到编辑器 |
| 图片预览 | WYSIWYG 模式下内联显示 |
| 相对路径 | 图片路径使用相对路径 `./assets/image-xxx.png` |

**图片存储规则：**
```
当前编辑文件: /path/to/note.md
粘贴图片自动保存为: /path/to/assets/boltmd-20260416-143200.png
插入到编辑器: ![](./assets/boltmd-20260416-143200.png)
```

#### F07: 主题

| 功能 | 描述 |
|------|------|
| 亮色主题 | 默认主题 |
| 暗色主题 | 深色背景 |
| 跟随系统 | 自动切换 |
| 快捷切换 | 快捷键或状态栏点击切换 |

### P1 — 重要（发布时最好有）

#### F08: 状态栏

显示信息：
- 当前文件名和路径
- 保存状态（已保存 / 未保存●）
- 字符数 / 单词数 / 行数
- 当前光标位置（行:列）
- 当前编辑模式
- 编码格式
- 主题切换按钮

#### F09: 工具栏（WYSIWYG 模式）

常用格式化按钮：
- 标题级别（H1-H6）
- 加粗、斜体、删除线、行内代码
- 有序列表、无序列表、任务列表
- 引用、分隔线
- 插入链接、插入图片
- 插入表格
- 插入代码块

可配置显示/隐藏（Ctrl+Shift+T）。

#### F10: 窗口管理

- 窗口大小和位置记忆（下次打开恢复）
- 自定义标题栏（显示文件名 + 保存状态）
- 最小化/最大化/关闭

#### F11: 自动保存

- 可配置开关（默认关闭）
- 防抖延迟可配置（默认 3 秒）
- 自动保存时状态栏短暂提示

#### F11.5: YAML Frontmatter

- 高亮显示 `---` 包裹的头部元数据
- WYSIWYG 模式下以特殊区块展示，不作为正文渲染

#### F12: 多标签页

| 功能 | 描述 | 快捷键 |
|------|------|--------|
| 新建标签页 | 新建空白标签 | Ctrl+N |
| 打开到新标签 | 打开文件时在新标签页显示 | Ctrl+O |
| 关闭标签页 | 关闭当前标签（有修改时提示保存） | Ctrl+W |
| 切换标签页 | 左右切换标签 | Ctrl+Tab / Ctrl+Shift+Tab |
| 跳转到第N个标签 | 直接跳转 | Ctrl+1~9 |
| 拖拽排序 | 鼠标拖拽调整标签顺序 | — |
| 标签状态 | 未保存的标签显示 ● 标记 | — |
| 右键菜单 | 关闭 / 关闭其他 / 关闭右侧 / 复制路径 | — |

**技术要点：**
- 每个标签页维护独立的编辑器实例和文件状态
- 非活跃标签页的编辑器实例可以销毁（只保留内容+光标状态），切换回来时重建
- 标签页状态（打开的文件列表、顺序、活跃标签）随配置持久化
- 同一文件不允许在多个标签页重复打开，已打开则跳转到对应标签

## MVP 不做的功能

| 功能 | 原因 | 计划版本 |
|------|------|---------|
| 数学公式 (KaTeX) | 需要额外依赖，使用场景有限 | v0.2 |
| 插件系统 | 需要设计插件 API | v0.2 |
| 文件树侧栏 | MVP 是文件编辑器不是项目管理器 | v0.3 |
| 云同步 | 需要后端服务 | v0.4+ |
| AI 功能 | 需要 API 集成 | v0.5+ |
| 多语言 | 先做好英文+中文 | v0.4 |
| 协作编辑 | 复杂度极高 | v1.0+ |
| 移动端 | 先做好桌面端 | v2.0 |
