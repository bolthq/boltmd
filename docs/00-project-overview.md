# BoltMD — 项目企划方案书

## 一句话定位

**BoltMD 是一个极速、轻量、现代化的 Markdown 文件编辑器。**

## 项目背景

### 为什么做这个

1. **现有工具太慢** — Typora 启动 3-5 秒，编辑长文档卡顿，基于 Electron 的方案普遍臃肿
2. **AI 时代 .md 文件爆发** — AI Agent、Skill 配置、知识库文档大量使用 Markdown 格式，需要一个专门的、足够快的编辑器
3. **开源作品积累** — 作为独立开发者的代表作品，积累开源社区资历

### 目标用户

- 程序员（日常编辑 README、文档、笔记）
- AI 工程师（编辑 agent skill、prompt、知识库文件）
- 技术写作者（博客、文档、教程）
- 任何需要快速打开 .md 文件的人

### 核心卖点

1. **极速** — 启动 <500ms，比 Typora 快 10 倍
2. **所见即所得** — Typora 级别的 WYSIWYG 编辑体验
3. **轻量** — 安装包 <15MB，内存占用 <100MB
4. **现代化 UI** — 简洁美观，支持亮色/暗色主题
5. **文件编辑器定位** — 不是笔记应用，是 .md 文件的默认打开方式

## 竞品定位

| | Notepad3 | **BoltMD** | Typora | Obsidian | VS Code |
|---|---------|-----------|--------|----------|---------|

## 技术方案

| 项目 | 选型 | 理由 |
|------|------|------|
| 应用框架 | Tauri 2.0 | 轻量（不打包 Chromium）、跨平台、Rust 后端 |
| 前端框架 | Vue 3 + TypeScript | 开发效率高、响应式系统适合编辑器 |
| WYSIWYG 引擎 | Tiptap (ProseMirror) | 成熟稳定、29k stars、MIT 协议 |
| 源码编辑器 | CodeMirror 6 | 大文件性能好、虚拟滚动、Markdown 扩展丰富 |
| Markdown 解析(Rust) | pulldown-cmark | 后端性能敏感操作备用 (v0.2+) |
| 样式方案 | TailwindCSS | 快速开发、主题切换方便 |
| 打包 | Tauri NSIS/WiX | Windows 安装包 + 文件类型关联 |

## 开源与商业

- **协议**：Apache 2.0
- **模式**：Open Core — 核心编辑器开源，未来高级功能（云同步、AI、协作）可作为付费插件
- **GitHub 仓库**：`boltmd`
